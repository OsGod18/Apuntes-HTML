Apuntes-HTML. README.

DESCRIPCION:

Este proyecto cuenta con una barra de navegación y tres secciones(inicio, apuntes generales y proyectos). Es una guía sobre fundamentos del lenguaje demarcado HTML. Su finalidad es probar nuevos estilos, animaciones y estructura web. Su contenido es poco y básico. Implementa tres tecnologías, HTML, CSS y JavaScript(solo animaciones). Cuenta con diversidad de imágenes-ejemplo y contenido explicado por el mismo desarrollador.

INSTALACION:

Descarga el archivo.zip "Apuntes HTML.zip". Ingresas a la carpeta extraída, buscas la carpeta "Sección principal" y presiona sobre el archivo "inicio.html". Entraras al inicio de los apuntes y navega por la pagina web.
