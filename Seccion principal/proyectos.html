Proyecto #1: Catalogo de libros resumidos

Este proyecto empezo como una estrategia por parte de Os. Le encantan los libros acerca del "Desarrollo personal". Siempre ha tenido la idea de que el mejor metodo de aprendizaje era la enseñanza, sino podias enseñarle un concepto de manera breve a una persona, significa que no aprendiste nada. Teniendo eso en cuenta, decidio resumir libros que ya habia leido, apuntar sus notas en un cuaderno y luego crear una plantilla web de cada libro y modificarlo en base a su portada y contenido. Este proyecto consta de una seccion principal donde se realiza una introduccion al proyecto y luego abre tres enlaces a tres diferentes libros que se encuentran disponibles con sus resumenes.

Proyecto #3: Apuntes HTML

Este proyecto nacio debido al metodo de aprendizaje de Os. Cuando el aprende un lenguaje de programacion o una herramienta tecnologica, se centraba en crear un documento donde realizaba "tutoriales". Explica cada concepto como si fuera a entregarle ese documento a otra persona para que se guiara y aprendiera. Aplico este metodo al momento de aprender HTML, y, para reforzar conocimiento, decidio volver a crear la pagina desde cero pero mas ambientado a una guia.

Proyecto #5: Catalogo virtual EPS

Este proyecto empezo por un trabajo universitario de Os y su respectivo grupo de trabajo. El proposito era investigar una problematica general y realizar una solucion con un software. Un miembro del grupo, planeto la problematica y Os decidio dar la solucion con este software. Al final, el proyecto fue un completo exito, tardo 15 horas en construirse y fue un paso revolucionario en la ruta de aprendizaje de Os. Este proyecto cuenta con una conexion de una base de datos(MySQL). Un puente estable(conexion a base de datos, Python, flask), y se compone de plantillas.
