¿Que es una pagina web?

Una pagina web es un documento digital que se accede a traves de internet usando un navegador. Este documento puede contener texto, imagenes, videos, enlaces y otros elementos interactivos. Cada pagina web tiene una direccion URL unica que permite acceder a ella desde cualquier lugar con conexion a internet.

Tipos de pagina web:

Las paginas web pueden clasificarse segun su funcion o contenido:

  • Estaticas: Presentan contenido fijo que no cambia para diferentes usuarios
  • Dinamicas: Su contenido se actualiza automaticamente o segun la interaccion del usuario, por ejemplo, redes sociales, tiendas en linea o blogs.
  • Informativas: Diseñadas para ofrecer datos, noticias o articulos educativos.
  • Comerciales: Su objetivo principal es la venta de productos o servicios.
  • Interactiva o de servicios: Permite acciones como registro de datos, consultas o transacciones electricas.

Las paginas web son esenciales actualmente porque permiten a las personas y empresas difundir informacion, comunicarse y ofrecer servicios a nivel global

HTML

Aunque existen diversas plataformas y softwares los cuales permiten crear paginas web dinamicas, en poco tiempo y de manera sencilla, es indiscutible tener conocimiento tecnico sobre el lenguaje demarcado HTML.

HTML o HyperText Markup Language(Lenguaje de Marcado de Hipertexto), es un lenguaje estandar utilizado para estructuras y presentar contenido en la web. No es un lenguaje de programacion ya que se abstiene del uso de logica de programacion. Esta herramienta define la estructura de una pagina web mediante etiquetas y atributos.

Las etiquetas que se usan al momento de crear una pagina web, pueden definir diferentes tipos de contenido, como párrafos, encabezados, imágenes, listas, enlaces, tablas, formularios, entre otros.

Historia de HTML

Ya que conocemos un concepto breve sobre este lenguaje estandar, debemos tambien conocer un poco de su historia.

HTML fue desarrollado a finales de la decada de 1980 y principios de los 90 por Tim Berners-Lee, un fisico y cientifico de la computacion britanico que trabajaba en el CERN(organizacion Europea para la Investigacion Nuclear) en Suiza. Esta herramienta fue creada con el proposito de compartir informacion de manera rapida, flexible, a distancia y organizada entre los investigadores de su epoca, dando origen a la World Wide Web(WWW).

Estructura de un documento HTML

Un documento.html sostiene una estructura la cual permite el desarrollo y presentacion de una pagina web. Esta estructura contiene dos etiquetas importantes para la organizacion de etiquetas de la pagina web; <body> Y <head>.

El head contiene informacion no visible para el usuario, como el titulo de la pagina, metaetiquetas(explicacion mas abajo), estilos CSS y logica de programacion Javascript. El body alberga el contenido de la pagina con la posibilidad de que el usuario pueda interactuar.

HTML semantico

Una pagina web esta formado por una estructura detallada la cual permite sostener y optimizar la busqueda de informacion, brindar una mejor experiencia de usuario y mantener un diseño accesible y estetico. Algunas secciones claras de una estructura web, son:

  1. Header(cabecera): Es la parte superior de la pagina web. Su objetivo es proporcionar identidad de marca y facilitar la navegacion. El header puede ser fijo o dinámico, permitiendo que el usuario lo vea aun al hacer scroll. Esta seccion incluye:
    • Logotipo de la marca, normalmente vinculado a la pagina principal
    • Menu de navegacion principal(enlace de secciones; inicio, productos, servicios y contacto.)
    • Llamadas a la accion(CTA), como botones de registrarse, comprar o contactar.
    • Barra de busqueda, informacion de contacto, o accesos rapidos(todo esto puede ser opcional).

Body(cuerpo): El cuerpo es la seccion central donde se incluyen los contenidos propios de cada pagina; elementos como imágenes, videos, banners, textos, formularios, sliders, carrousels, y cualquier contenido interactivo que aporte valor al usuario. Su estructura puede ser en bloque o por secciones semanticas:
    • <main>: aqui se ubica el contenido principal
    • <article>: bloques de contenido independiente, como un post de blog o ficha de producto.
    • <section>: agrupa secciones relacionadas dentro de la pagina
    • <aside>: contenido complementario o lateral, como enlaces relacionados, publicidad o widgets
    • <div> y <span>: envolturas no semanticas para diseño u organizacion CSS/JS

    • Informacion alternativa: enlaces a politicas, terminos de uso, cookies y sitemap.
    • Certificaciones: logo de colaboradores, derechos de autor, y enlaces a otras secciones del sitio web

Etiquetas basicas de HTML

En este apartado, conoceras las etiquetas basicas que forman el contenido de una pagina web. Estas son esenciales ya que permiten crear cosas sencillas pero necesarias para la presentacion de informacion.

La etiqueta <title> permite modificar el titulo que se ubica en la pestaña de la pagina web. Esta etiqueta se ubica dentro de la etiqueta(bloque) <head> del documento HTML. Es util para asignar el titulo de la pagina web.

Etiquetas decorativas de HTML

Existen diferentes etiquetas las cuales cumplen la funcion de marcar palabras, asignar contenido importante o estructurar el texto para lograr tener una mayor comprension de la informacion. Estas etiquetan son las siguientes:

<br>

La etiqueta de line break(separador de linea) se utiliza para insertar un salto de línea en el texto. Es util en casos donde las líneas deben dividirse de manera específica, como en poemas o direcciones. Sin embargo, para diseño y espaciado general, es mejor usar estilos CSS. Es un elemento vacío, por lo que no tiene etiqueta de cierre.

LISTAS

Las listas son estructuras que permiten organizar informacion de manera clara y legible. Se dividen en listas no ordenadas, listas ordenadas y listas de descripcion.

Importante: para poder crear la estructura de la lista, debemos tener una etiqueta la cual tomara el rol de "contenedor", todo esto con el fin de indicarle al navegador que el contenedor es una lista. Esta etiqueta "contenedor" encapsula los elementos que se incluiran en la lista con otro tipo de etiqueta.

Listas no ordenadas

Las listas no ordenadas muestran los elementos con viñetas(•), sin un orden relevante. Se definen con <ul> (unordered list) y cada elemento con <li> (list item).

Atributos de ancle

Los atributos son datos que se añaden a la etiqueta para dar informacion adicional o indicar las acciones que debe realizar el navegador respecto al atributo. Diferentes tipos de atributos(existen mas pero veras unos importantes):

Imagenes

La posibilidad de insertas imagenes en una pagina web, ocurre con el uso de la etiqueta <img>(image). Esta etiqueta es un elemento vacio(no tiene contenido, ni etiqueta de cierre) y se define principalmente mediante atributos:

  • src: El atributo source(fuente), indica la ruta o URL de la imagen.
  • alt: El atributo alternative(alternativo), muestra un texto alternativo en caso de que la imagen no haya podido cargar correctamente
  • width: El atributo widht(ancho) define el tamaño sentido horizontal de la imagen en pixeles o porcentaje
  • height: El atributo height(altura) define el tamaño sentido vertical de la imagen en pixeles o porcentaje
  • title: El atributo title(titulo) muestra un titulo(nosotros se lo asignamos) al pasar el cursor sobre la imagen.
